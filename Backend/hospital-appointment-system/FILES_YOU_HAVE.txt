================================================================================
🔧 COMMON ISSUES & QUICK FIXES
================================================================================

"Cannot connect to server"
  Problem:   Backend not running
  Solution:  Run ./mvnw spring-boot:run in project directory

"Port 8080 already in use"
  Problem:   Another application using port
  Solution:  Change server.port in application.properties

"Validation failed on email"
  Problem:   Invalid email format in request
  Solution:  Use format: user@example.com

"Invalid token error"
  Problem:   Token expired or wrong
  Solution:  Login again to get fresh token

"Database connection error"
  Problem:   MySQL not running (if configured)
  Solution:  Start MySQL server

Full troubleshooting: See TESTING_GUIDE.md section "Troubleshooting"
